# Changelog

## [0.3.1] - 2026-03-15

### Added
- Initial release of OpenAI-compatible API adapter
- Support for any OpenAI-compatible LLM service (local or remote)
- Configuration for baseUrl, model, temperature, maxTokens, topP, timeout
- Support for optional API key authentication
- Environment variable configuration
- Custom prompt template support
- Bootstrap prompt for fresh sessions
- Environment testing and diagnostics
- Support for Claude, Gemini, GPT-4, and local LLM models
